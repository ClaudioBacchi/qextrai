# qExtrai

Prototipo navigabile della Fase 1: fondazione grafica e interfaccia dimostrativa per qExtrai.

## Comandi

```bash
npm install
npm.cmd run dev
npm.cmd run lint
npm.cmd run test
npm.cmd run build
npm.cmd run desktop:dev
npm.cmd run desktop:build
```

L'applicazione non include backend, OCR, lettura PDF reale o chiamate esterne.
